Pour rentrer vos identifiants de bdd aller dans config, ouvrir le fichier dbconf.php.dist,puis dans host mettez votre adresse de bdd et dans dbname le nom de la base de donnée, puis dans user et password vos identifiants puis renommer le fichier en dbconf.php

Votre vhost dois pointer sur ../public/index.php

La structure sql de la base vide ce trouve dans le fichier technowStructureVide.sql
Une structure avec des données test se trouve dans le fichier technow.sql
